     <metadata>
          <title>setAttributeNodeNS05</title>
          <creator>NIST</creator>
          <description> The "setAttributeNodeNS(newAttr)" method raises an "WRONG_DOCUMENT_ERR DOMException if the "newAttr" was created from a different document than the one that created this document. Retrieve the first emp:address and attempt to set a new attribute node. The new attribute was created from a document other than the one that created this element, therefore a WRONG_DOCUMENT_ERR DOMException should be raised. This test uses the "createAttributeNS(newAttr)" method from the Document interface. </description>
          <contributor>Mary Brady</contributor>
          <date qualifier='created'>2001-08-17</date>
          <subject resource='http://www.w3.org/TR/DOM-Level-2-Core/core#xpointer(id('ID-258A00AF')/constant[@name='WRONG_DOCUMENT_ERR'])'/>
          <subject resource='http://www.w3.org/TR/DOM-Level-2-Core/core#ID-ElSetAtNodeNS'/>
          <subject resource='http://www.w3.org/TR/DOM-Level-2-Core/core#xpointer(id('ID-ElSetAtNodeNS')/raises/exception[@name='DOMException']/descr/p[substring-before(.,':')='WRONG_DOCUMENT_ERR'])'/>
</metadata>
