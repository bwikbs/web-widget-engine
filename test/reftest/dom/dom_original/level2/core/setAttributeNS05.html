          <description> The "setAttributeNS(namespaceURI,qualifiedName,value)" method adds a new attribute. If an attribute with the same local name and namespace URI is already present on the element, its prefix is changed to be the prefix part of the "qualifiedName", and its vale is changed to be the "value" paramter. null value if no previously existing Attr node with the same name was replaced. Add a new attribute to the "emp:address" element. Check to see if the new attribute has been successfully added to the document. The new attribute "<newValue>" contains markup and therefore is escaped by the implementation. </description>
