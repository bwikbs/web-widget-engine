     <metadata>
          <title>setAttributeNodeNS01</title>
          <creator>NIST</creator>
          <description> The "setAttributeNodeNS(namespaceURI,qualifiedName,value)" method for an attribute causes the DOMException NO_MODIFICATION_ALLOWED_ERR to be raised if the node is readonly. Obtain the children of the THIRD "gender" element. The elements content is an entity reference. Try to set an attribute in the entity reference by executing the "setAttributeNodeNS(newAttr)" method. This causes a NO_MODIFICATION_ALLOWED_ERR DOMException to be thrown. </description>
