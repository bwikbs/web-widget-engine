          <description> The method createAttributeNS raises a NAMESPACE_ERR if the qualifiedName has a prefix that is "xml" and the namespaceURI is different from "http://www.w3.org/XML/1998/namespace". Check if the NAMESPACE_ERR exception is thrown. </description>

</metadata>
