          <description> The "feature" parameter in the isSupported(feature,version)" method is the name of the feature and the version is the version number of the feature to test. CORE is a legal value for the feature parameter (Test for CORE, upper case). Legal values for the version parameter are 1.0 and 2.0 (Test for 1.0). Retrieve the root node of the DOM document by invoking the "getDocumentElement()" method. This should create a node object on which the "isSupported(feature,version)" method is invoked with "feature" equal to "CORE" and the version equal to 1.0. The method should return a boolean "true". </description>
