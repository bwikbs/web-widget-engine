          <creator>NIST</creator>
          <description> The "removeAttributeNS(namespaceURI,localName)" method for an attribute causes the DOMException NO_MODIFICATION_ALLOWED_ERR to be raised if the node is readonly. Obtain the children of the THIRD "gender" element. The elements content is an entity reference. Try to remove an attribute from the entity reference by executing the "removeAttributeNS(namespaceURI,localName)" method. This causes a NO_MODIFICATION_ALLOWED_ERR DOMException to be thrown. </description>
