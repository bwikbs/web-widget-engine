          <title>documentGetElementsByTagNameValue</title>
          <creator>NIST</creator>
          <description> The "getElementsByTagName(tagName)" method returns a NodeList of all the Elements with a given tagName in a pre-order traversal of the tree. Retrieve the entire DOM document and invoke its "getElementsByTagName(tagName)" method with tagName equal to "name". The method should return a NodeList that contains 5 elements. The FOURTH item in the list is retrieved and output. </description>
          <contributor>Mary Brady</contributor>
          <date qualifier='created'>2001-08-17</date>
          <subject resource='http://www.w3.org/TR/1998/REC-DOM-Level-1-19981001/level-one-core#ID-A6C9094'/>
</metadata>
