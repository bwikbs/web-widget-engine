     <metadata>
          <title>elementRemoveAttributeNodeNoModificationAllowedErr</title>
          <creator>NIST</creator>
          <description> The "removeAttributeNode(oldAttr)" method for an attribute causes the DOMException NO_MODIFICATION_ALLOWED_ERR to be raised if the node is readonly. Obtain the children of the THIRD "gender" element. The elements content is an entity reference. Try to remove the "domestic" attribute from the entity reference by executing the "removeAttributeNode(oldAttr)" method. This causes a NO_MODIFICATION_ALLOWED_ERR DOMException to be thrown. </description>
          <contributor>Mary Brady</contributor>
