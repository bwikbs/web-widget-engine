          <title>attrEffectiveValue</title>

          <description> If an Attr is explicitly assigned any value, then that value is the attributes effective value. Retrieve the attribute named "domestic" from the last child of of the first employee and examine its nodeValue attribute. This test uses the "getNamedItem(name)" method from the NamedNodeMap interface. </description>

</metadata>

     <var name='addressList' type='NodeList'/>
     <var name='testNode' type='Node'/>

     <var name='domesticAttr' type='Attr'/>
